CSS Grid Elements

1. CSS Grid란?

CSS Grid는 행(Row)과 열(Column)이 있는 그리드 기반 레이아웃 시스템입니다. 웹 페이지 전체 구조를 2차원(행+열)으로 설계할 수 있습니다.

기본 조건
• 그리드 레이아웃을 사용하려면 부모 요소가 display: grid 또는 inline-grid 이어야 합니다.
• 그리드 컨테이너의 직접 자식 요소는 자동으로 Grid 아이템이 됩니다.

2. grid-template-columns란?

grid-template-columns는 열(Column)의 개수와 각 열의 너비(비율/크기)를 지정하는 속성입니다. 

grid-template-columns: 값 값 값;
  • 값의 개수 = 열의 개수입니다.
  • 각 값은 해당 열의 너비를 의미합니다.

3. fr 단위란?

fr(fraction)은 그리드에서 남은 공간을 나누는 비율 단위입니다.
예를 들어 1fr 1fr 1fr 은 남은 공간을 1 : 1 : 1 비율로 나눕니다.

한 줄 요약
grid-template-columns: 1fr 1fr 1fr; → 남은 공간을 1 : 1 : 1 비율로 나눠 3열을 만듭니다.

6. 핵심 정리

  • display:grid → Grid 컨테이너 생성
  • grid-template-columns → 열 개수와 열 너비를 지정합니다.
  • 값 개수 = 열 개수입니다.
  • fr은 남은 공간을 나누는 비율 단위입니다.
  • 1fr 1fr 1fr → 남은 공간을 1:1:1로 나눕니다.
