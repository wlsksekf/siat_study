Flexbox 정렬 : justify-content

1. Flexbox란?

Flexbox(Flexible Box)는 요소들을 한 줄 또는 여러 줄로 배치하고 정렬하기 위한 CSS 레이아웃 방식입니다. 특히 가로 정렬과 세로 정렬을 매우 쉽게 처리할 수 있습니다.

2. Flex 컨테이너 만들기

.flex-container {
  display: flex;
}

display:flex; 를 적용한 요소를 Flex 컨테이너라고 부르며, 그 안의 자식 요소들은 Flex 아이템이 됩니다.

3. justify-content란?

justify-content 속성은 Flex 컨테이너 안에서 아이템을 가로 방향(주축)으로 정렬하는 방법을 지정합니다.
(단, flex-direction: row; 가 기본값일 때)

4. justify-content 속성값

값설명
flex-start아이템들을 왼쪽부터 정렬합니다.
flex-end아이템들을 오른쪽부터 정렬합니다.
center아이템들을 가운데 정렬합니다.
space-between첫 번째는 왼쪽 끝, 마지막은 오른쪽 끝에 배치하고 사이 간격을 동일하게 합니다.
space-around각 아이템 주변에 동일한 여백을 둡니다. (양 끝 여백은 절반)
space-evenly모든 간격을 완전히 동일하게 배치합니다.

5. 기본 예제

.flex-container {
  display: flex;
  justify-content: center;
}

위 예제는 justify-content: center; 로 설정되어 이미지들이 가로 방향으로 가운데 정렬됩니다.

6. justify-content 값 변경 실습 + 결과

아래는 justify-content 값을 변경했을 때의 실제 실행 결과입니다.

1) justify-content: flex-start

→ 아이템들이 왼쪽부터 정렬됩니다.

2) justify-content: flex-end

→ 아이템들이 오른쪽부터 정렬됩니다.

3) justify-content: center

→ 아이템들이 가운데 정렬됩니다.

4) justify-content: space-between

→ 첫 번째는 왼쪽 끝, 마지막은 오른쪽 끝에 배치되고 사이 간격이 동일합니다.

5) justify-content: space-around

→ 각 요소 주변에 동일한 여백이 생기며, 양 끝 여백은 절반입니다.

6) justify-content: space-evenly

→ 모든 간격이 완전히 동일하게 배치됩니다.

7. 정리

  • display:flex → Flex 컨테이너 생성
  • justify-content → 가로(주축) 방향 정렬
  • flex-direction 기본값은 row
  • 값에 따라 아이템 배치가 달라집니다.
