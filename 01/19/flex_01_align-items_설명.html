Flexbox 정렬 : align-items

1. align-items란?

align-items 속성은 Flex 컨테이너의 교차축(cross axis, 세로 방향)으로 아이템들을 어떻게 정렬할지 결정합니다.
(단, flex-direction: row; 가 기본값일 때)

기억하기
justify-content : 가로(주축) 정렬
align-items : 세로(교차축) 정렬

2. align-items 속성값

값설명
flex-start아이템을 컨테이너의 위쪽(시작점)으로 정렬합니다.
flex-end아이템을 컨테이너의 아래쪽(끝점)으로 정렬합니다.
center아이템을 세로 가운데로 정렬합니다.
baseline아이템을 텍스트 기준선(baseline)에 맞춰 정렬합니다.
stretch아이템을 컨테이너 높이에 맞게 늘립니다(아이템에 높이가 지정되지 않은 경우).

3. 기본 예제

.flex-container {
  display: flex;
  align-items: flex-end;
}

위 예제는 align-items: flex-end; 로 설정되어 이미지들이 컨테이너 아래쪽으로 정렬됩니다.

4. align-items 값 변경 실습 + 결과

아래는 align-items 값을 바꿨을 때의 실제 실행 결과입니다.

1) align-items: flex-start

→ 아이템들이 위쪽부터 정렬됩니다.

2) align-items: flex-end

→ 아이템들이 아래쪽부터 정렬됩니다.

3) align-items: center

→ 아이템들이 세로 가운데 정렬됩니다.

4) align-items: baseline

baseline은 텍스트 기준선을 맞추는 정렬입니다.

5) align-items: stretch

stretch는 아이템의 높이가 고정되어 있지 않으면(기본값 auto) 컨테이너 높이에 맞춰 늘어납니다.
이미지는 기본적으로 늘어나지 않는 경우가 많아, 실습에서는 이미지가 늘어나는 모습을 보기 쉽도록 스타일을 일부 조정했습니다.

→ 아이템이 컨테이너 높이에 맞춰 늘어납니다(조건에 따라 달라질 수 있습니다).

5. 정리

  • align-items는 세로(교차축) 정렬을 담당합니다.
  • flex-direction 기본값은 row 이므로 교차축은 세로입니다.
  • flex-start / center / flex-end 로 위·가운데·아래 정렬을 쉽게 할 수 있습니다.
  • baseline은 텍스트 기준선을 맞추는 정렬입니다.
  • stretch는 아이템의 높이가 지정되지 않았을 때 컨테이너 높이에 맞춰 늘어납니다.
