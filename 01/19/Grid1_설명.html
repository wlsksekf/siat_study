CSS Grid Layout : grid-template-areas

1. CSS Grid란?

CSS Grid는 행(Row)과 열(Column)을 기반으로 웹 페이지 전체 레이아웃을 설계하는 2차원 레이아웃 시스템입니다. float이나 position 없이도 복잡한 화면 구조를 쉽게 만들 수 있습니다.

✔ Flexbox → 한 줄(1차원) 정렬 중심
✔ Grid → 행과 열(2차원) 레이아웃 설계 중심

2. Grid 컨테이너 만들기

.grid-container {
  display: grid;
}

display:grid; 를 적용한 요소를 Grid 컨테이너라고 부르며, 그 안의 자식 요소들은 Grid 아이템이 됩니다.

3. grid-template-areas란?

grid-template-areas는 화면 레이아웃을 문자 이름으로 설계하는 방식입니다. 각 영역에 이름을 붙이고, 그 이름을 배치하여 페이지 구조를 만듭니다.

4. 기본 문법

grid-template-areas:
  "header header header"
  "menu main right"
  "footer footer footer";

따옴표 한 줄이 하나의 행(Row)을 의미합니다.

5. 영역 이름 지정

.item1 { grid-area: header; }
.item2 { grid-area: menu; }
.item3 { grid-area: main; }
.item4 { grid-area: right; }
.item5 { grid-area: footer; }

각 Grid 아이템에 grid-area 이름을 지정하면 grid-template-areas 구조에 맞춰 자동 배치됩니다.

6. 전체 예제 코드

.grid-container {
  display: grid;
  grid-template-areas:
    "header header header header header header"
    "menu main main main right right"
    "menu footer footer footer footer footer";
  gap: 10px;
}

7. 실행 결과

Header
Menu
Main
Right
Footer

→ 이름으로 설계한 레이아웃이 화면에 그대로 배치됩니다.

8. 레이아웃 구조 시각화

"header header header header header header"
"menu   main   main   main   right  right"
"menu   footer footer footer footer footer"

각 단어가 화면의 영역을 의미합니다.

9. 정리

  • Grid는 2차원(행 + 열) 레이아웃 시스템입니다.
  • grid-template-areas는 레이아웃을 이름으로 설계합니다.
  • grid-area로 각 요소를 지정하면 자동 배치됩니다.
  • 복잡한 웹 페이지 구조를 쉽게 만들 수 있습니다.
