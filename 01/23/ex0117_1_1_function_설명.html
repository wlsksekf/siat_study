JavaScript Function (함수)

함수 개념, 선언/표현식, 호출 방법, 매개변수·return, 즉시 실행 함수(IIFE)

1) 함수란?

함수(Function)는 여러 줄의 코드를 묶어 이름(또는 변수)에 저장해 두었다가, 필요할 때 호출하여 실행하는 구조입니다. 함수를 사용하면 코드 재사용과 유지보수가 쉬워집니다.

재사용 구조화 유지보수 가독성
함수는 JavaScript에서 자료형이 function인 값입니다. 따라서 변수에 저장되고, 전달될 수 있습니다.

2) 함수 기본 형태

2-1. 함수 선언식(Function Declaration)

이름이 반드시 있으며, 실행 전에 함수가 미리 등록되어 선언 위치와 관계없이 같은 범위 안에서는 먼저 호출할 수 있습니다. 

function add(a, b) {
  return a + b;
}

alert(add(3, 5)); // 8
함수 선언식으로 만든 함수는 자바스크립트가 코드를 실행하기 전에 미리 등록해 둡니다(호이스팅).
따라서 같은 범위 안에서는 함수 선언보다 호출이 앞에 있어도 정상적으로 실행됩니다.

2-2. 함수 표현식(Function Expression)

함수의 이름이 없는 함수를 익명함수라고 합니다.
함수를 변수에 저장한 뒤 변수를 통해 호출해야 하며 함수는 할당이 끝난 후에만 사용할 수 있습니다. 

let sub = function(a, b) {
  return a - b;
};

alert(sub(10, 4)); // 6

2-3. 이름이 있는 함수 표현식(Named Function Expression)

함수에 이름을 붙일 수 있으며, 디버깅 시 함수 이름이 표시되어 유용한 경우가 있습니다.

let a = function bar() {
  return 3;
};

alert(a()); // 3
함수 표현식은 변수에 함수가 할당되기 전에는 호출할 수 없습니다.
즉, 선언보다 호출이 먼저 오면 오류가 발생합니다.

3) 함수 호출 방법(Calling a Function)

함수는 함수 이름 뒤에 괄호 ()를 붙여 호출합니다. 괄호 안에는 함수가 필요로 하는 값(인수, argument)을 전달할 수 있습니다.

3-1. 기본 호출: 함수명()

function hello() {
  alert("안녕하세요!");
}

hello(); // 함수 호출

3-2. 인수(argument) 전달: 함수명(값, 값)

함수 정의에서 받는 쪽이 매개변수(parameter), 호출할 때 넣는 값이 인수(argument)입니다.

function add(a, b) {   // a, b는 매개변수(parameter)
  return a + b;
}

alert(add(3, 5)); // 3, 5는 인수(argument)

3-3. 반환값을 변수에 저장해서 사용

function multiply(x, y) {
  return x * y;
}

let result = multiply(4, 3); // 반환값 저장
document.write(result);      // 12

3-4. 이벤트에서 호출(버튼 클릭 등)

HTML 이벤트 속성 또는 addEventListener로 함수를 연결하여 호출할 수 있습니다.

<button onclick="sayHi()">인사</button>

<script>
function sayHi() {
  alert("반갑습니다!");
}
</script>
<button id="btn">클릭</button>

<script>
function onClickBtn() {
  alert("이벤트로 호출되었습니다.");
}

document.querySelector("#btn").addEventListener("click", onClickBtn);
</script>
주의: addEventListener("click", onClickBtn) 처럼 함수 이름만 전달해야 합니다. onClickBtn() 처럼 괄호를 붙이면 즉시 실행되어 이벤트가 아니라 실행 결과가 전달됩니다.

4) 매개변수(parameter)와 반환값(return)

4-1. 매개변수(parameter)

함수에 전달되는 입력값을 받는 변수입니다.

function greet(name) {
  alert(name + "님, 환영합니다.");
}

greet("홍길동");

4-2. 반환값(return)

함수 실행 결과를 돌려주는 키워드입니다. return이 없으면 undefined가 반환됩니다.

function multiply(x, y) {
  return x * y;
}

let result = multiply(4, 3);
alert(result); // 12
return을 만나면 함수 실행이 즉시 종료됩니다.

5) 즉시 실행 함수(IIFE-Immediately Invoked Function Expression)

즉시 실행 함수는 정의하자마자 바로 실행되는 함수입니다. 한 번만 실행하고 끝내는 초기화 코드에 자주 사용됩니다. 

(function () {
  alert("즉시 실행합니다.");
})();

(함수 정의) 를 괄호로 감싸서 “값”으로 만든 뒤, 바로 ()를 붙여 호출합니다.

6) typeof로 함수 자료형 확인

함수는 자료형이 function으로 확인됩니다.

function test() {}

console.log(typeof test); // function

7) 함수 선언식 vs 함수 표현식 비교

구분 형태 호출 방법 특징
함수 선언식 function name(){} name() 호이스팅 영향으로 script 태그 내용을 읽기 전에 선언적 함수부터 읽혀 선언 이전에도 호출이 가능한 경우가 많습니다.
함수 표현식 let f = function(){} f() 변수에 저장된 후 호출됩니다. 선언보다 호출이 먼저 오면 오류가 발생합니다.
즉시 실행 함수 (function(){})() 자동 실행 정의 즉시 1회 실행됩니다. 초기화 코드에 자주 사용됩니다.

8) 실습 예제: 입력 → 처리 → 출력

prompt로 입력받고, 함수로 처리한 뒤, 결과를 화면에 출력합니다.

function showNumber() {
  let n = prompt("숫자를 입력하세요", "숫자");
  document.body.innerHTML += "<p>입력한 숫자 = " + n + "</p>";
}

showNumber();

9) 정리

  • 함수는 코드를 묶어 저장하고 필요할 때 호출하여 실행됩니다.
  • 함수 선언식은 function name(){} 형태로 작성됩니다.
  • 함수 선언식은 script 처리 시 선언적 함수가 먼저 읽혀 선언 이전에도 호출 가능한 경우가 많습니다.
  • 함수 표현식은 함수를 변수에 저장한 뒤 변수명() 형태로 호출됩니다.
  • 즉시 실행 함수는 정의하자마자 한 번 실행됩니다.
  • typeof 결과는 function 입니다.
