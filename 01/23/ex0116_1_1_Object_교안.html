JavaScript 객체(Object) 기초

실습 결과는 개발자 도구(F12) → Console에서 확인합니다.

1. 객체란?

객체(Object)는 여러 개의 값을 하나로 묶어서 관리하는 자료구조입니다.
객체는 여러 개의 속성(Property)으로 구성됩니다.

Property = Key(이름) + Value(값)
Key는 속성의 이름이며, Value는 그 이름에 연결된 값입니다.
Value가 함수이면 메서드(Method)라고 부릅니다.

2. 객체 내용 접근 방식

(1) 점 연산자(.) 이용

객체 내부의 속성(Property)과 메서드(Method)는 점 연산자(.)를 사용하여 접근합니다.

접근 문법 형식

객체.Key      // 속성(Property) 접근
객체.Key()    // 메서드(Method) 호출

즉, 객체 이름 뒤에 마침표(.)와 속성 이름(Key)을 작성하여 해당 속성의 Value에 접근합니다.

예를 들어 saram 객체에 name 이라는 Key가 있다면 다음과 같이 접근합니다. 

saram.name

✔ 점 연산자 예제

// 객체 생성
let saram = {};

// 속성 추가
saram.name = "나최고";
saram.age = 25;

// 메서드 추가
saram.print = function(){
  console.log("이름:", saram.name);
  console.log("나이:", saram.age);
};

// 속성 접근
console.log(saram.name);

// 메서드 호출
saram.print();

(2) 대괄호([]) 표기법

  • 객체["속성이름"] 형태로 접근합니다.
  • 속성 이름(Key)이 변수에 저장된 경우 반드시 대괄호 표기법을 사용합니다.
saram["name"] = "나최고";
saram['name'] = "나최고";
언제 대괄호 표기법을 쓰나요?
속성 이름이 변수에 들어있거나,
공백·특수문자가 포함된 경우 대괄호 표기법을 사용합니다.

3. 객체 생성 방법

방법 예시 설명
new Object() let saram = new Object(); Object 생성자를 이용한 객체 생성
객체 리터럴 { } let saram = {}; 가장 많이 사용하는 객체 생성 방식

4. 객체 속성 추가

객체 생성 후 속성을 추가하므로 동적 속성 추가라고 합니다.

let saram = {};
saram.name = "나최고";

5. 속성 삭제

delete saram.name;

6. for...in 문으로 객체 순회

for...in 문은 객체의 모든 속성 이름(Key)을 하나씩 꺼내 반복하는 문법입니다.

✔ p에는 속성 이름(Key)이 문자열로 저장됩니다.
✔ 따라서 반드시 객체[p] 형태로 값(Value)에 접근합니다. 
let output = "";

// for...in → 객체의 속성 이름(Key) 순회
for (let p in saram) {
  output += `saram.${p} = ${saram[p]}`;
}

document.body.innerHTML = output;

실행 결과

saram.name = 나최고
saram.age = 25
saram.ki = 170.2
saram.blood = A
saram.address = 서울시 강남구
정리
for...in → 객체의 속성 이름(Key) 반복
객체[p] → 해당 속성의 값(Value) 접근
