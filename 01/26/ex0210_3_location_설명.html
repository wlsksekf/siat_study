JavaScript location 객체

1. location 객체란?

location 객체는 브라우저의 주소 표시줄(URL)과 관련된 정보를 담고 있는 객체입니다. 현재 열려 있는 페이지의 주소 정보를 읽거나, 새로운 페이지로 이동할 때 사용됩니다.

2. 주요 속성

속성 설명 예시
href 문서의 전체 URL 주소입니다. http://localhost:8080/index.html
protocol 사용 중인 프로토콜 종류입니다. http:, https:
hostname 호스트 이름(도메인)입니다. localhost
host 호스트 이름 + 포트 번호입니다. localhost:8080
port 포트 번호입니다. 8080
pathname 서버 내 문서의 경로입니다. /index.html

3. location 정보 출력 예제

<script>
let output = "";

for(let key in location){
  output += key + " : " + location[key] + "\n";
}

alert(output);
</script>

for...in 문을 이용하여 location 객체의 모든 속성과 값을 출력합니다.

4. 페이지 이동 기능

function movePage(){
  location.href = "https://www.google.com";
}

function replacePage(){
  location.replace("https://www.google.com");
}

function reloadPage(){
  location.reload();
}

✔ location.href → 새로운 페이지로 이동합니다.
✔ location.replace() → 뒤로가기 기록 없이 이동합니다.
✔ location.reload() → 현재 페이지를 새로고침합니다.

5. 정리

  • location 객체는 현재 페이지의 URL 정보를 담고 있습니다.
  • 주소 정보를 읽거나 변경하여 페이지 이동이 가능합니다.
  • 웹 페이지 네비게이션 제어에 사용됩니다.
