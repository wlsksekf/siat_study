DOM(Document Object Model)과 이벤트

자바스크립트로 HTML 요소를 제어하는 기초 개념

1. DOM이란?

DOM(Document Object Model)은 웹 브라우저가 HTML 문서를 객체(Object) 구조로 변환한 것입니다.

  • HTML 태그 하나 → 자바스크립트 객체 하나
  • 태그의 포함 관계 → 트리 구조 (DOM 트리)
  • DOM 트리의 최상위 객체 → document

2. 문서 객체 생성 방식

  • 정적 생성 : HTML에 작성된 태그를 브라우저가 읽으며 자동 생성
  • 동적 생성 : 자바스크립트로 새로운 태그를 만들어 추가

3. 이벤트(Event)

사용자의 동작이나 브라우저 상태 변화에 반응하는 기능입니다.

  • 버튼 클릭
  • 키보드 입력
  • 페이지 로드 완료

이벤트 구성 요소

  • 이벤트 타입 : load, click, keydown ...
  • 이벤트 리스너 : onload, onclick ...
  • 이벤트 핸들러 : 실제 실행되는 함수

4. window.onload 의미

HTML 페이지의 모든 태그가 화면에 로드된 후 실행되는 함수를 등록합니다. 

window.onload = function() {
   // 페이지 로드 완료 후 실행
};

5. getElementById()

DOM 트리에서 특정 id를 가진 요소를 찾아 반환합니다. 

let headerObject = document.getElementById('header');
  • id가 존재 → 해당 객체 반환
  • id가 없으면 → null 반환

6. 전체 예제 코드

<script>
window.onload = function() {
   let headerObject = document.getElementById('header');
   headerObject.style.color = 'red';
};
</script>

<h1 id="header">HEADER</h1>

9. DOM 트리와 노드(Node) 구성 요소

DOM 트리는 HTML 문서를 브라우저가 객체(Object) 계층 구조(트리)로 변환한 결과입니다. 즉, 태그의 포함 관계가 부모-자식 관계로 연결되어 트리 형태가 됩니다.

DOM 트리란?

  • document가 최상위(루트)입니다.
  • HTML 태그는 요소 노드(Element Node)로 만들어집니다.
  • 태그 안의 글자는 텍스트 노드(Text Node)로 분리되어 자식으로 연결됩니다.

포인트: 화면에 보이는 “글자”도 DOM에서는 별도의 노드(텍스트 노드)로 취급됩니다.
