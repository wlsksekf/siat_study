1. 파이썬 자료형 분류

① 기본 자료형

자료형예설명
int10정수
float3.14실수
boolTrue논리값
str"Hello"문자열

② 컬렉션 자료형

자료형예특징
list[1,2,3]순서 O, 수정 O
tuple(1,2,3)순서 O, 수정 X
set{1,2,3}중복 X
dict{"a":1}Key-Value

③ 변경 가능성 (Mutable / Immutable)

핵심 개념 Python에서 모든 값은 객체(Object)입니다. 객체는 값 자체를 변경할 수 있는지 여부에 따라 Mutable(변경 가능) 과 Immutable(변경 불가능) 로 나뉩니다.
구분자료형특징
Mutable list, dict, set 객체 내부 값을 직접 수정 가능
Immutable int, float, bool, str, tuple 객체 내부 값 수정 불가 → 새 객체 생성

✔ Mutable 예시


a = [1,2,3]
a[0] = 100
print(a)      # [100, 2, 3]
리스트 객체 내부 값이 직접 변경됩니다.

✔ Immutable 예시


s = "hello"
s = s + "!"
print(s)      # hello!
문자열은 수정 불가 → 새 문자열 객체가 생성됩니다.

✔ 메모리 관점


x = 10
y = x
x = 20
  • 10 과 20 은 각각 별도의 int 객체
  • x 는 10 → 20 객체로 참조 변경
  • 기존 10 객체는 y 가 계속 참조
정리
Mutable 객체는 "객체 내부 값 변경"
Immutable 객체는 "새 객체를 만들어 참조 변경"

따라서 Python 변수는 값을 담는 것이 아니라 객체를 가리키는 이름표(참조) 입니다.

④ 특수 자료형

핵심 개념 특수 자료형은 특정 목적을 위해 제공되는 파이썬의 내장(Built-in) 자료형입니다.
자료형예설명
NoneType None 값이 없음을 의미 (null 개념)
range range(1, 5) 연속된 숫자 시퀀스를 생성

✔ NoneType


x = None
print(x)         # None
print(type(x))   # <class 'NoneType'>
변수에 아직 값이 없음을 표현할 때 사용

✔ range


r = range(1, 5)
print(list(r))   # [1, 2, 3, 4]
반복문에서 숫자 시퀀스를 효율적으로 생성
정리
None → 값 없음 표현
range → 숫자 시퀀스 생성
정리
파이썬의 모든 값은 객체이며 용도에 따라 다양한 자료형으로 분류됩니다.
