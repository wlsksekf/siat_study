자바스크립트 Array(배열)

1) Array란 무엇인가요

Array(배열)은 여러 개의 값을 순서대로 저장하는 자료구조입니다. 배열은 하나의 변수에 여러 값을 담을 수 있어 반복문과 함께 자주 사용됩니다.

정리: 배열은 “여러 값을 순서대로 저장하는 상자”로 이해됩니다.

2) 인덱스(index)란 무엇인가요

인덱스는 배열의 각 값을 구분하는 번호입니다. 인덱스는 항상 0부터 시작됩니다. 

const fruits = ["사과", "바나나", "포도", "딸기"];
console.log(fruits[0]); // "사과"가 출력됩니다
console.log(fruits[3]); // "딸기"가 출력됩니다
주의: 길이가 4인 배열의 마지막 인덱스는 3입니다.

3) 배열 구조(그림)

아래 그림은 배열이 “인덱스(위) + 값(아래)” 형태로 저장된 것처럼 이해할 수 있도록 시각화한 것입니다.

확인: 개발자 도구 콘솔(Console)에서 fruits, fruits.length를 직접 확인해 보세요.

4) 배열 생성 방법

방법 ① 대괄호 사용

const numbers = [10, 20, 30];

방법 ② Array 생성자 사용

const numbers = new Array(10, 20, 30);
정리: 실무에서는 대괄호([ ]) 방식이 더 많이 사용됩니다.

5) 배열 길이(length)

length는 배열에 저장된 요소의 개수를 나타냅니다. 

const fruits = ["사과", "바나나", "포도", "딸기"];
console.log(fruits.length); // 4가 출력됩니다

6) 배열 값 접근(읽기/쓰기)

읽기

console.log(fruits[1]); // "바나나"가 출력됩니다

쓰기(변경)

fruits[1] = "키위";
console.log(fruits); // ["사과","키위","포도","딸기"]가 출력됩니다
주의: 존재하지 않는 인덱스에 값을 넣으면 배열이 늘어날 수 있습니다. 
fruits[10] = "멜론";
console.log(fruits.length); // 11이 출력됩니다 (중간은 비어 있을 수 있습니다)

7) 반복문으로 배열 순회하기

for문

for (let i = 0; i < fruits.length; i++) {
  console.log("index:", i, "value:", fruits[i]);
}

for...of문

for (const v of fruits) {
  console.log(v);
}
정리: 값만 순서대로 처리할 때는 for...of문이 자주 사용됩니다.
