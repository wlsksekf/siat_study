정규식과 RegExp(Regular Expression) 객체

1. 정규식(Regular Expression)이란?

정규 표현식은 문자열에서 특정 문자 조합(패턴)을 찾거나, 문자열 형식을 검사(검증)하기 위한 규칙입니다.
자바스크립트에서 정규식은 RegExp 객체로 동작하며, 리터럴(/.../)로 만들더라도 내부적으로는 RegExp 객체입니다.

예) /abc/ instanceof RegExp 결과는 true입니다.

2. 정규식 작성 방법

① 정규식 리터럴 방식

슬래시(/) 사이에 패턴을 작성합니다.

// 예: 주민번호 형태(예시) 6자리-7자리
let reg = /^[0-9]{6}-[0-9]{7}$/;

② RegExp 생성자 방식

문자열로 패턴을 전달하여 정규식 객체를 생성합니다.
주의: 문자열 안에서는 백슬래시(\)가 “이스케이프 문자”로 쓰이므로, \d 같은 표현을 넣으려면 \\d처럼 “백슬래시를 한 번 더” 써야 합니다. 

// ^[0-9]{6}-[0-9]{7}$ 는 백슬래시가 없어 그대로 사용 가능
let reg1 = new RegExp("^[0-9]{6}-[0-9]{7}$");

// \d 를 쓰고 싶다면 문자열에서는 \\d 로 작성해야 합니다.
let reg2 = new RegExp("^\\d+$"); // 숫자 1개 이상
핵심 요약
- 리터럴: /\d+/ (OK)
- 생성자: new RegExp("\\d+") (OK)
- 생성자에 "\d+"라고 쓰면 \d가 제대로 전달되지 않습니다.

3. 정규식 주요 기호(메타 문자)

기호 의미 예시
^ 문자열의 시작을 의미합니다. /^1/ : 1로 시작
$ 문자열의 끝을 의미합니다. /1$/ : 1로 끝
[ ] 대괄호 안의 문자 중 하나를 선택합니다. /[a-z]/, /[0-9]/
[0-9], \d 숫자 1개를 의미합니다. (\d는 숫자 축약 표현) /\d\d/ : 숫자 2개 연속
{n} 바로 앞 패턴을 정확히 n번 반복합니다. /^\d{3}$/ : 숫자 3자리만
{n,m} 바로 앞 패턴을 n~m번 반복합니다. /^\d{2,4}$/ : 2~4자리
{n,} 바로 앞 패턴을 n번 이상 반복합니다. /^\d{2,}$/ : 2자리 이상
+ 바로 앞 패턴이 1개 이상 반복합니다. /\d+/ : 숫자가 1개 이상 포함
* 바로 앞 패턴이 0개 이상 반복합니다. /a*/ : a가 없어도 OK
? 바로 앞 패턴이 0 또는 1개 존재합니다. /colou?r/ : color/colour
(A|B) 그룹으로 묶어 A 또는 B를 의미합니다.
(묶음 단위로 OR 조건을 적용합니다.) 		/^(1|2)$/ : 전체가 1 또는 2

4. test() 메서드

정규식.test(문자열) 형태로 사용하며, 패턴에 맞으면 true, 아니면 false를 반환합니다. 

let reg = /^[0-9]+$/;
reg.test("123"); // true
reg.test("12a"); // false
주의 1) test()는 값이 문자열이 아니어도 “문자열로 변환” 후 검사합니다.
예) /^\d+$/.test(100) 은 내부적으로 "100"으로 바꾼 뒤 검사하므로 true입니다.
