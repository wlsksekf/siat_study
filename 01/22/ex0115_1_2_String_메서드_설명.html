자바스크립트 String 객체

문자열을 다루는 String 객체와 주요 메서드를 학습합니다.

1) String 객체란?

String 객체는 문자(텍스트) 데이터를 다루기 위한 자바스크립트 내장 객체입니다. 문자열은 작은따옴표(' '), 큰따옴표(" "), 백틱(` `)으로 생성됩니다.

정리: String 객체는 문자열 처리에 필요한 다양한 메서드를 제공합니다.

2) 문자열 생성 방법

let str1 = "Hello";
let str2 = 'JavaScript';
let str3 = `문자열 템플릿`;

3) 문자열 길이 (length)

let msg = "Hello World";
console.log(msg.length); // 11 출력됩니다

4) 문자열 인덱스 접근

문자열도 배열처럼 인덱스로 각 문자에 접근할 수 있습니다. 

let word = "ABC";
console.log(word[0]); // A 출력됩니다
console.log(word[1]); // B 출력됩니다

5) 문자열 대소문자 변환

let text = "JavaScript";

console.log(text.toUpperCase()); // "JAVASCRIPT"
console.log(text.toLowerCase()); // "javascript"

6) 문자열 검색 메서드

메서드 설명
indexOf() 문자의 위치를 찾습니다. 없으면 -1 반환됩니다
includes() 문자 포함 여부를 true/false로 반환합니다 
let msg = "Hello JavaScript";

console.log(msg.indexOf("Java")); // 6 출력됩니다
console.log(msg.includes("Script")); // true 출력됩니다

7) 문자열 자르기

메서드 설명
slice(start, end) 문자열 일부를 잘라 반환합니다
substring(start, end) slice와 비슷하지만 음수 인덱스를 허용하지 않습니다 
let text = "Hello JavaScript";

console.log(text.slice(0,5)); // Hello 출력됩니다
console.log(text.substring(6,16)); // JavaScript 출력됩니다

8) 문자열 치환 (replace)

let msg = "I like JavaScript";
console.log(msg.replace("JavaScript","HTML")); 
// "I like HTML" 출력됩니다

9) 공백 제거 (trim)

let input = "   hello   ";
console.log(input.trim()); // "hello" 출력됩니다

10) 문자열 분리와 결합

let fruits = "사과,바나나,포도";
let arr = fruits.split(",");
console.log(arr); // ["사과","바나나","포도"]

let result = arr.join(" / ");
console.log(result); // "사과 / 바나나 / 포도"

11) 실전 예제

let email = prompt("이메일 입력");

if(email.includes("@")) {
  document.body.innerHTML = "올바른 이메일입니다.";
} else {
  document.body.innerHTML = "이메일 형식이 아닙니다.";
}
