HTML 기본 구조

HTML 문서는 요소(element)와 속성(attribute)으로 구성되며, 문서의 시작에는 DOCTYPE 선언이 포함됩니다.

1. HTML 문서란 무엇인가요

HTML(HyperText Markup Language)은 웹 문서의 구조를 만드는 언어입니다.
HTML 문서는 태그(tag)로 표기하는 요소(element)와, 요소에 추가 정보를 주는 속성(attribute)으로 이루어집니다.

HTML 문서는 보통 다음 구성으로 이루어집니다.
DOCTYPE html 요소 head 요소 body 요소

2. DOCTYPE이란 무엇인가요

DOCTYPE은 문서를 구성하는데 사용한 규약(표준)을 알려주는 선언입니다.
문서가 어떤 HTML 규격을 따르는지 브라우저에게 알려주는 역할을 합니다.

웹 문서에 DOCTYPE 정보가 없으면 브라우저는 문서의 규약을 정확히 알 수 없어
웹 문서를 잘못 처리할 수 있습니다.

2-1. HTMLDOCTYPE

<!DOCTYPE html>

위 선언은 “이 문서는 HTML규약을 따릅니다” 라는 의미입니다.

3. <html> 요소의 역할

<html> 요소는 문서 전체를 감싸며, 이 안에 문서 내용이 들어갑니다.
일반적으로 다음 두 영역이 포함됩니다.

  • <head> 요소: 문서에 대한 정보(메타데이터)가 포함됩니다.
  • <body> 요소: 브라우저 화면에 보이는 실제 내용이 포함됩니다.

4. <head> 요소에 들어가는 것들

<head> 요소에는 “문서 제목, 문자 인코딩, CSS/JS 연결 정보” 같은 문서 정보가 들어갑니다.

4-1. <title> 요소

<title> 요소는 문서 제목을 나타냅니다.
브라우저 탭 제목으로 표시됩니다.

4-2. <meta> 요소

<meta> 요소는 문자 인코딩 정보 또는 검색엔진을 위한 정보 등을 설정합니다.

HTML5에서 가장 기본이 되는 인코딩 설정은 다음과 같습니다.
<meta charset="UTF-8"> 로 설정됩니다.

4-3. 기타 head 요소

  • <link> 요소: 외부 CSS 파일 등을 연결합니다.
  • <script> 요소: 자바스크립트를 작성하거나 외부 JS 파일을 연결합니다.
  • <style> 요소: 문서 내부에 CSS를 작성합니다.

5. <body> 요소에 들어가는 것들

<body> 요소에는 브라우저 화면에 출력되는 내용이 들어갑니다.
예를 들어 제목, 단락, 목록, 표, 이미지 등이 포함됩니다.

6. 요소(element)와 속성(attribute)

6-1. 요소(element)란 무엇인가요

요소는 웹 문서를 구성하는 기본 단위입니다.
일반적으로 시작 태그 + 내용 + 종료 태그로 이루어집니다. 

<p>안녕하세요</p>
<h1>제목입니다</h1>

6-2. 속성(attribute)란 무엇인가요

속성은 요소에 추가 정보를 주기 위해서 사용됩니다.
속성은 보통 시작 태그 안에 작성되며, 이름과 값의 쌍으로 구성됩니다. 

<img src="cat.png" alt="고양이 사진">
<a href="https://example.com">이동하기</a>
  • 이름과 값 사이에는 = 문자가 사용됩니다.
  • 값은 " " 또는 ' '로 감싸서 작성됩니다.
  • 요소는 여러 속성을 가질 수 있으며, 속성은 공백으로 구분됩니다.

7. 태그(tag)와 단독 태그

태그는 문서에서 요소를 표시하기 위해 붙이는 꼬리표입니다.
태그에는 시작 태그와 종료 태그가 있을 수 있으며, 내용이 없는 요소는 단독 태그로 사용됩니다.

7-1. 복합 태그(시작+종료가 있는 태그)

<title>문서 제목</title>
<div>내용</div>

7-2. 단독 태그(내용이 없는 태그)

단독 태그는 내용이 없기 때문에 종료 태그가 없습니다.

<img src="cat.png" alt="고양이">
<meta charset="UTF-8">
<link rel="stylesheet" href="style.css">
<br>
예: img, meta, link, br, hr, input 등으로 사용됩니다.

8. 주석(comment)

주석은 브라우저 화면에는 보이지 않지만, 문서 작성자가 메모를 남기기 위해 사용됩니다. 

<!-- 여기는 주석입니다. 브라우저 화면에는 표시되지 않습니다. -->

9. 특수문자(예약 문자)

HTML에서 <, > 같은 문자는 태그를 표현하는데 사용되므로 그대로 쓰면 문제가 생길 수 있습니다.
이때는 엔티티(entity)로 작성합니다.

표현하려는 문자 HTML 엔티티 설명
< &lt; 태그 시작 기호이므로 엔티티로 작성합니다.
> &gt; 태그 끝 기호이므로 엔티티로 작성합니다.
& &amp; 엔티티 시작 기호이므로 엔티티로 작성합니다.
" &quot; 속성 값에서 큰따옴표가 필요할 때 사용합니다.

10. 문자 코드(인코딩)

웹 문서는 문자 인코딩을 지정해야 글자가 깨지지 않습니다.
가장 많이 사용하는 방식은 UTF-8이며, head에 다음과 같이 작성합니다. 

<meta charset="UTF-8">

11. HTML 기본 구조 전체 예제

아래 예제는 HTML기본 구조를 모두 포함한 예제입니다.
그대로 복사해서 .html 파일로 저장하면 실행됩니다. 

<!DOCTYPE html>
<html lang="ko">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>나의 첫 HTML 문서</title>

  <!-- 외부 CSS 연결 예시 -->
  <link rel="stylesheet" href="style.css">

  <!-- 내부 CSS 예시 -->
  <style>
    body { font-family: Arial, sans-serif; }
  </style>
</head>
<body>
  <h1>안녕하세요</h1>
  <p>이 문서는 HTML 기본 구조 예제입니다.</p>

  <!-- 이미지(단독 태그) 예시 -->
  <img src="cat.png" alt="고양이 사진">

  <!-- 자바스크립트 예시 -->
  <script>
    console.log("HTML 문서가 로드되었습니다.");
  </script>
</body>
</html>
정리하면, HTML 문서는 DOCTYPE → html → head/body 순서로 작성되며,
내용은 요소로 구성되고, 요소의 추가 정보는 속성으로 표현됩니다.

정리

  • DOCTYPE은 문서 규약을 알려줍니다.
  • head는 문서 정보를 포함합니다.
  • body는 화면에 보이는 내용을 포함합니다.
  • 요소는 태그로 표현됩니다.
  • 속성은 시작 태그 안에서 이름=값으로 작성됩니다.
