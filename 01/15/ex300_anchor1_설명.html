하이퍼링크(Hyperlink)

1. 하이퍼링크란?

하이퍼링크(hyperlink)는 HTML 문서에서 다른 문서로 이동할 수 있도록 연결하는 기능입니다. 텍스트나 이미지에 웹 페이지 주소(URL)를 연결하여 클릭하면 해당 페이지로 이동됩니다. 하이퍼링크는 줄여서 링크(link)라고 부릅니다.

2. 하이퍼링크 기본 문법

<a href="이동할 주소">표시할 텍스트</a>
속성설명
href이동할 웹 페이지 주소(URL)를 지정합니다.
title마우스를 올리면 말풍선 설명이 표시됩니다.

3. 절대 주소(외부 사이트) 이동 예제

다음 예제는 다른 웹 사이트로 이동하는 절대 주소 하이퍼링크 예제입니다. 

<a href="https://www.w3c.or.kr" title="W3C 대한민국 사무국">
  W3C 사무국(KOR) 방문
</a>

<a href="https://www.w3schools.com/" title="배우러 가요">
  https://www.w3schools.com/
</a>

<a href="https://entertain.naver.com/home" title="TV연예">
  네이버 연예
</a>

4. 실제 실행 예제

  • W3C 사무국(KOR) 방문
  • https://www.w3schools.com/
  • 네이버 연예

각 링크를 클릭하면 지정된 웹 사이트로 이동됩니다.

5. 새 창에서 열기

<a href="https://www.naver.com" target="_blank">
네이버 새 창 열기
</a>

target="_blank" 속성을 사용하면 링크가 새 탭(새 창)에서 열립니다.

6. 정리

  • <a> 태그로 하이퍼링크를 만듭니다.
  • href 속성에 이동할 주소를 작성합니다.
  • title 속성은 말풍선 설명이 표시됩니다.
  • target="_blank" 는 새 창에서 열립니다.
