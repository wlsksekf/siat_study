복합 대입 연산자 (Compound Assignment Operator)

복합 대입 연산자는 산술 연산자와 할당 연산자(=)를 결합한 연산자입니다.

같은 변수를 기준으로 연산한 뒤 그 결과를 다시 자기 자신에게 저장할 때 사용합니다.

1. 실행 예제 코드


let a = 10;
let b = 4;

// 일반 산술 연산 + 할당
a = a + b; 
console.log(a); // 14

// 복합 대입 연산자
a += b; 
console.log(a); // 18

a -= b; 
console.log(a); // 14

a *= b; 
console.log(a); // 56

a /= b; 
console.log(a); // 14

a %= b; 
console.log(a); // 2

실행 결과는 브라우저 개발자 도구의 Console 창에서 확인할 수 있습니다.

2. 복합 대입 연산자 표

복합 연산자 의미 일반 표현
+= 덧셈 후 저장 a = a + b
-= 뺄셈 후 저장 a = a - b
*= 곱셈 후 저장 a = a * b
/= 나눗셈 후 저장 a = a / b
%= 나머지 후 저장 a = a % b

3. 정리

  • 복합 대입 연산자는 연산과 할당을 동시에 수행한다.
  • 코드를 더 간결하고 읽기 쉽게 만든다.
