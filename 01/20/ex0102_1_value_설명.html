자바스크립트에서 값(Value)이란?

자바스크립트에서 값(Value)이란 변수에 저장되거나, 표현식의 결과로 만들어지는 실제 데이터를 의미합니다.

1. 값의 기본 개념

  • 변수는 값을 담는 그릇입니다.
  • 값은 프로그램에서 처리의 대상이 되는 데이터입니다.
let x = 10;
  • x → 변수
  • 10 → 값

2. 값의 종류 (자료형 기준)

자바스크립트의 모든 값은 자료형(type)을 가집니다.

2-1. 기본형 값 (Primitive Value)

자료형 설명
primitivenumber 숫자 값 (10, 3.14)
primitivestring 문자열 값 ("hello")
primitiveboolean 논리 값 (true, false)
primitiveundefined 값이 할당되지 않음
primitivenull 의도적으로 값이 없음 
let a = 100;
let b = "JS";
let c = true;

2-2. 객체형 값 (Object Value)

종류 설명
objectobject 여러 값을 묶은 객체
objectarray 순서가 있는 값의 집합
objectfunction 실행 가능한 코드 블록 
let obj = { name: "kim" };
let arr = [1, 2, 3];
let fn  = function () {};

3. 값과 자료형의 관계

자바스크립트에서는 변수가 타입을 가지는 것이 아니라, 값이 타입을 가진다. 
let x = 10;      // number 값
x = "hello";     // string 값
  • 변수 x는 그대로 유지됩니다.
  • 저장된 값이 바뀌면 자료형도 함께 바뀝니다.

4. 값은 언제 만들어지는가?

값은 다음 경우에 생성됩니다.

  1. 직접 작성한 리터럴
    리터럴(literal)은 사람이 이해할 수 있는 문자 또는 약속된 기호로 값을 직접 표현하는 표기법입니다.

    자바스크립트 리터럴 종류

    리터럴 종류 예제 설명
    정수(숫자) 리터럴 10 숫자 값을 직접 표현
    문자열 리터럴 "hi" 큰따옴표 또는 작은따옴표로 감싼 문자 데이터
    불리언 리터럴 true, false 참(true) 또는 거짓(false) 값
    객체 리터럴 {} 키-값 쌍으로 구성된 객체 표현
    배열 리터럴 [] 여러 값을 순서대로 저장하는 자료구조
    함수 리터럴 function() {} 함수를 직접 정의하는 표현식
    정규 표현식 리터럴 /[A-Z]+/g 문자열 패턴을 검사·검색하기 위한 표현식
  2. 연산 결과 
    5 + 3   // 8
  3. 함수 반환값 
    return 100;

5. 정리

자바스크립트에서 값이란 변수에 저장되거나 연산의 결과로 생성되는 실제 데이터이며, 모든 값은 고유한 자료형을 가진다.
핵심 요약
  • 값은 변수에 저장되거나 표현식의 결과로 만들어지는 데이터
  • 자바스크립트는 값이 타입을 가진다(동적 타입)
  • 값은 리터럴, 연산 결과, 함수 반환값으로 생성될 수 있다
