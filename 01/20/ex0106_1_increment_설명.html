증감 연산자 (++ / --)

증감 연산자는 변수의 값을 1씩 증가(++), 또는 1씩 감소(--)시키는 연산자입니다.

핵심
  • ++x : 먼저 증가(감소)한 뒤 값을 사용
  • x++ : 먼저 값을 사용한 뒤 증가(감소)

1. 실행 예제 코드

let x = 5;

console.log(++x); // 6 (전위 증가)
console.log(x++); // 6 (후위 증가: 출력 후 증가)
console.log(x);   // 7

실행 결과는 브라우저 개발자 도구의 Console 창에서 확인할 수 있습니다.

2. 전위 증가 연산자 (++x)

전위 증가 연산자는 값을 먼저 1 증가시킨 후, 증가된 값을 사용합니다. 

let x = 5;
console.log(++x); // x가 먼저 6이 되고, 6이 출력됨

3. 후위 증가 연산자 (x++)

후위 증가 연산자는 현재 값을 먼저 사용한 다음, 그 후에 값이 1 증가합니다. 

let x = 6;
console.log(x++); // 먼저 6이 출력되고, 출력 이후 x는 7이 됨

4. 실행 흐름 정리

단계 코드 출력값 x 값 변화
1 초기값 - x = 5
2 console.log(++x) 6 x: 5 → 6 (먼저 증가)
3 console.log(x++) 6 x: 6 → 7 (출력 후 증가)
4 console.log(x) 7 x = 7
중요
++x, x++는 모두 실행 결과로 하나의 값을 반환하므로 표현식(Expression)입니다.

5. 추가: 감소 연산자(--)

감소 연산자도 증가 연산자와 동일한 규칙을 가집니다. 

let y = 5;

console.log(--y); // 4 (전위 감소)
console.log(y--); // 4 (후위 감소: 출력 후 감소)
console.log(y);   // 3
