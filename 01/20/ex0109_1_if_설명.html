조건문 if / if-else

조건문(Conditional Statement)은 조건식의 결과에 따라 실행할 코드를 결정하는 문장입니다.

if문은 조건식이 true 또는 false로 평가되어 실행 흐름을 제어합니다.

1. if문

if문은 조건식이 true일 때만 중괄호 안의 코드를 실행합니다. 

if (조건식) {
  실행할 문장;
}

예시

let score = 75;

if (score >= 60) {
  console.log("합격입니다.");
}
  • score >= 60 → true 이므로 실행
  • false이면 아무 동작도 하지 않음

2. if - else문

if-else문은 조건이 true 인 경우와 false 인 경우를 모두 처리합니다. 

if (조건식) {
  조건이 true일 때 실행
} else {
  조건이 false일 때 실행
}

예시

let score = 75;

if (score >= 80) {
  console.log("우수한 성적입니다.");
} else {
  console.log("좀 더 노력하세요.");
}
  • score ≥ 80 → "우수한 성적입니다."
  • 그 외 → "좀 더 노력하세요."

3. Truthy / Falsy 와 if문

if문의 조건식에는 Boolean 값뿐 아니라 Truthy / Falsy 규칙이 적용됩니다. 

if (값) {
  // 값이 Truthy 이면 실행
} else {
  // 값이 Falsy 이면 실행
}
Falsy 값
false, 0, "", null, undefined, NaN
Truthy 값
Falsy를 제외한 모든 값 (예: "hi", 1, [], {}, function(){})

예시

let input = "";

if (input) {
  console.log("입력값이 있습니다.");
} else {
  console.log("입력값이 없습니다.");
}

4. 정리

  • if문은 조건이 true 일 때만 실행됩니다.
  • if-else문은 true / false 경우를 모두 처리합니다.
  • 조건식은 Truthy / Falsy 규칙에 따라 평가됩니다.
  • 조건문은 프로그램의 실행 흐름을 제어합니다.
조건문은 조건식의 결과에 따라 실행할 코드를 선택하는 제어문입니다.
