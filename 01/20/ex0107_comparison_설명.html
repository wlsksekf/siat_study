비교 연산자 (Comparison Operator)

비교 연산자는 두 값을 비교하여 true 또는 false를 반환하는 연산자입니다.

비교 연산식 전체는 실행 결과로 true 또는 false를 반환하므로 표현식(Expression)입니다.

1. 실행 예제 코드

let a = 10;
let b = 5;
let c = "10";

console.log(a == c);   // true
console.log(a === c);  // false
console.log(a > b);    // true

실행 결과는 브라우저 개발자 도구의 Console 창에서 확인할 수 있습니다.

2. 비교 연산자 종류

연산자 의미 설명
== 같다 값만 비교 (자료형 자동 변환)
=== 완전히 같다 값과 자료형을 모두 비교
!= 다르다 값이 다른지 비교
!== 완전히 다르다 값 또는 자료형이 다른지 비교
> 크다 왼쪽 값이 오른쪽 값보다 큰지 비교
< 작다 왼쪽 값이 오른쪽 값보다 작은지 비교
>= 크거나 같다 왼쪽 값이 크거나 같은지 비교
<= 작거나 같다 왼쪽 값이 작거나 같은지 비교

3. == 와 === 의 차이

10 == "10"   // true
10 === "10"  // false
  • == : 값만 비교 → 자동 형 변환 발생
  • === : 값 + 자료형 비교 → 형 변환 없음
실무에서는 예측 가능한 코드를 위해 ===, !== 사용을 권장합니다.

4. 핵심 정리

  • 비교 연산자는 true / false 를 반환한다.
  • 비교식 전체는 표현식(Expression)이다.
  • == 는 자동 형 변환을 수행한다.
  • === 는 값과 자료형을 함께 비교한다.
