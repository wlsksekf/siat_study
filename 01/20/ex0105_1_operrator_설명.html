자바스크립트 연산자(Operator)

이 문서는 연산자(Operator)의 개념과 구성 요소, 주요 종류를 정리합니다.

연산자는 값(피연산자)에 대해 계산·비교·논리 판단·저장 등의 작업을 수행하는 기호입니다.

1. 연산자란 무엇인가?

연산자(Operator)는 값(피연산자)을 대상으로 연산을 수행하여 새로운 값을 만들거나(true/false 포함) 저장하는 기호입니다.

2. 연산자 구성 요소

아래 예시에서 연산의 대상과 기호를 구분해 봅니다.

3 + 5
요소 이름 설명
3, 5 피연산자(operand) 연산의 대상이 되는 값
+ 연산자(operator) 연산을 수행하는 기호
3 + 5 표현식(expression) 실행 결과로 하나의 값이 나오는 코드

3. 연산자의 역할 예시

연산자는 다양한 종류가 있으며, 목적에 따라 결과가 달라집니다.

역할 대표 연산자 예시 결과
계산산술 +, -, *, /, %, ** 10 / 4 2.5
비교조건 판단 >, <, >=, <=, ===, !== 10 > 4 true
논리조건 결합 &&, ||, ! true && false false
저장할당 =, +=, -=, *= ... x += 1 x = x + 1
선택조건 연산 ?: score >= 60 ? "합격" : "불합격" "합격" 또는 "불합격"

4. 예제: 연산자가 표현식을 만든다

연산자와 피연산자가 결합하면 표현식이 되고, 표현식은 실행 결과로 값이 나옵니다. 

let x = 10 + 5;
// 10 + 5 는 산술 연산 표현식이며 결과는 15

x = x + 1;
// x + 1 이 먼저 계산되고(표현식 → 값), 그 결과가 x에 저장됩니다.
정리
  • 연산자: 작업을 수행하는 기호
  • 피연산자: 연산의 대상(값)
  • 표현식: 실행 결과로 하나의 값이 나오는 코드
