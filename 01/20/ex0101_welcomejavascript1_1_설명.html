자바스크립트 코드 작성 위치

자바스크립트 코드는 HTML 문서 안에서 세 가지 위치에 작성할 수 있습니다.

1. HTML 태그의 이벤트 속성에 작성

HTML 태그의 이벤트 리스너 속성 (onclick, onload, onchange, onkeyup, onsubmit 등)에 직접 작성합니다. 

<img src="Koala.jpg" onclick="this.src='Desert.png'">
사용자가 이미지를 클릭하면 이미지 파일이 다른 그림으로 변경됩니다.

2. <script> 태그 안에 작성

HTML 문서 내부의 <script> 태그에 자바스크립트 코드를 작성합니다. 

<script>
  alert("자바스크립트 안녕~!!!");
</script>
페이지가 로드되면 알림창이 실행됩니다.

3. 외부 자바스크립트 파일에 작성

자바스크립트 코드를 별도의 파일(.js)에 작성한 뒤 HTML 문서에서 불러옵니다. 

<script src="welcome.js"></script>
여러 HTML 문서에서 같은 스크립트를 재사용할 수 있어 유지보수와 관리가 편리합니다.

4. 정리

  • 이벤트 속성 → 간단한 동작 처리
  • <script> 태그 → 문서 내부 스크립트
  • 외부 .js 파일 → 재사용성과 유지보수에 유리
자바스크립트 코드는 HTML 내부 또는 외부 파일에서 실행될 수 있습니다.
