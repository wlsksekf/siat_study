자바스크립트 변수 & 자료형

식별자 규칙, 변수 선언, typeof로 자료형 확인, undefined 정리합니다.

1) 식별자(Identifier) 규칙
식별자는 변수/함수/객체/속성 등에 이름을 붙일 때 사용하는 단어입니다.
  • 첫 글자는 알파벳, _, $만 가능합니다.
  • 예약어(break, case, catch, if, for, return 등)는 사용할 수 없습니다.
  • 숫자는 첫 글자를 제외하고 사용 가능합니다. (예: user1)
  • 특수문자는 _와 $만 허용합니다.
  • 공백을 포함할 수 없습니다.
  • 대소문자를 구분합니다. (name과 Name은 다릅니다)
올바른 예 잘못된 예
alpha, _alpha, $alpha, Alpha, user1 1alpha(숫자 시작), user name(공백), @name(특수문자), break(예약어)
2) 식별자 종류 (변수/속성/함수/메서드)
  • 괄호 없음 → 변수(단독) / 속성(다른 식별자와 함께)
  • 괄호 있음 → 함수(단독) / 메서드(다른 식별자와 함께)
input                // 변수
Array.length         // 속성(length)
alert('Hello World') // 함수
Math.abs(-273)       // 메서드(abs)
함수는 단독 호출 가능하고, 메서드는 보통 객체 뒤에 점(.)으로 붙어서 사용됩니다.
3) 변수 선언 & 자료형 (typeof)
변수 선언은 let, const를 사용합니다. 대소문자 구분
자바스크립트는 값이 저장되는 순간 자료형이 결정되는 동적 타입 언어입니다. 
/* 변수 선언 방식 */
let 변수명;
변수명 = 값;

/* 또는 */
let 변수명 = 값;

/* typeof로 자료형 확인 */
typeof 변수명
typeof(변수명)
