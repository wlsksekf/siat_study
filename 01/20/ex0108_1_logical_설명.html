논리 연산자 (Logical Operators)

논리 연산자는 true / false 값을 이용하여 조건을 판단할 때 사용하는 연산자입니다.

논리 연산식은 실행 결과로 하나의 값을 반환하므로 표현식(Expression)입니다.

1. 논리 연산자 종류

연산자 이름 설명
&& AND 두 값이 모두 true 일 때 true
|| OR 하나라도 true 이면 true
! NOT 논리값을 반대로 변환

2. 기본 동작

true && true   // true
true && false  // false

false || false // false
true || false  // true

!true          // false
!false         // true

3. 단락 평가 (Short-circuit Evaluation)

자바스크립트의 논리 연산자는 항상 true / false만 반환하지 않고 마지막으로 평가된 값을 반환합니다.

  • A && B → A가 Falsy 이면 A 반환, 아니면 B 반환
  • A || B → A가 Truthy 이면 A 반환, 아니면 B 반환
0 && 10     // 0
10 && 20    // 20

0 || 10     // 10
100 || 10   // 100

4. Truthy / Falsy 정리 표

구분 개념 예시 값 Boolean 변환 결과
Falsy 조건식에서 false로 평가되는 값 false false
0, -0
"" (빈 문자열)
null
undefined
NaN
Truthy Falsy를 제외한 모든 값 1, -1, 0.1 true
"0", "false"
[] (빈 배열)
{} (빈 객체)
function(){}

5. 활용 예

let userName = input || "손님";

input 값이 없으면 기본값 "손님"이 저장됩니다.

6. 정리

  • 논리 연산자는 조건 판단에 사용된다.
  • AND, OR 연산자는 단락 평가를 수행한다.
  • 논리 연산식은 마지막으로 평가된 값을 반환한다.
  • Truthy / Falsy 개념과 함께 사용된다.
