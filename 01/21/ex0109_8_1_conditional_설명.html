자바스크립트 조건 연산자 (Conditional Operator)

if문을 한 줄로 표현하는 삼항 연산자의 개념과 사용법을 학습합니다.

1) 조건 연산자란

조건 연산자는 조건에 따라 두 값 중 하나를 선택하는 연산자입니다. 형태가 조건 ? 참일 때 값 : 거짓일 때 값 으로 구성되므로 삼항 연산자 라고도 불립니다.

정리: 조건 연산자는 if문을 간단히 한 줄로 표현할 때 사용됩니다.

2) 기본 문법

조건식 ? 값1 : 값2;
  • 조건식이 true이면 값1이 선택됩니다.
  • 조건식이 false이면 값2가 선택됩니다.

3) if문과 비교

if문 방식

let score = 80;
let result;

if (score >= 60) {
  result = "합격";
} else {
  result = "불합격";
}

console.log(result);

조건 연산자 방식

let score = 80;
let result = (score >= 60) ? "합격" : "불합격";

console.log(result);
정리: 같은 동작을 더 간단한 문법으로 표현할 수 있습니다.

4) 숫자 판별 예제

let num = 7;

let msg = (num % 2 === 0) ? "짝수" : "홀수";

console.log(msg); // 홀수 출력됩니다

5) prompt 입력과 함께 사용

let age = Number(prompt("나이를 입력하세요"));

let result = (age >= 20) ? "성인" : "미성년자";

document.body.innerHTML = result + " 입니다.";

6) 중첩 조건 연산자

조건 연산자는 중첩해서 사용할 수 있지만 가독성이 떨어지므로 간단한 경우에만 사용됩니다. 

let score = 85;

let grade = (score >= 90) ? "A" :
            (score >= 80) ? "B" :
            (score >= 70) ? "C" : "D";

console.log(grade); // B 출력됩니다
주의: 중첩 조건 연산자는 복잡해지면 if문이 더 읽기 쉽습니다.

7) 조건 연산자 요약

항목 설명
형태 조건 ? 참값 : 거짓값
용도 간단한 if~else 대체
장점 코드가 간결해집니다
주의점 중첩 시 가독성이 떨어집니다

8) 체크 문제

  • if문을 조건 연산자로 바꿔보세요.
let x = 15;
let msg;

if (x > 10) {
  msg = "크다";
} else {
  msg = "작다";
}

9) 한 줄 요약

✔ 조건 연산자는 if문을 한 줄로 표현합니다
✔ 조건 ? 참값 : 거짓값 구조입니다
✔ 간단한 조건 처리에 사용됩니다
