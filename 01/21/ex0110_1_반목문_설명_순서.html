자바스크립트 반복문 (Loops)

for문과 while문의 실행 순서를 포함한 반복문 교안

1) for 반복문 기본 구조

for (초기값; 조건식; 증감식) {
   실행문;
}

for문 수행 순서

단계설명
① 초기값 실행반복문 시작 시 단 한 번 실행됩니다.
② 조건식 검사조건이 true이면 반복문이 실행됩니다.
③ 실행문 수행반복할 코드가 실행됩니다.
④ 증감식 실행변수 값을 증가 또는 감소시킵니다.
⑤ 다시 조건식 검사조건이 false가 될 때까지 ②~④가 반복됩니다.
정리: for문 순서 → 초기값 → 조건식 → 실행문 → 증감식 → 조건식 → … 반복

예제

for (let i = 1; i <= 3; i++) {
  console.log(i);
}
출력 결과 → 1 2 3 ---

2) while 반복문 기본 구조

while (조건식) {
   실행문;
}

while문 수행 순서

단계설명
① 조건식 검사조건이 true이면 반복문이 시작됩니다.
② 실행문 수행반복할 코드가 실행됩니다.
③ 다시 조건식 검사false가 될 때까지 반복됩니다.
정리: while문 순서 → 조건식 → 실행문 → 조건식 → … 반복
주의: 조건식이 항상 true이면 무한 반복이 발생합니다.

예제

let i = 1;
while (i <= 3) {
  console.log(i);
  i++;
}
출력 결과 → 1 2 3 ---

3) for문 vs while문 수행 순서 비교

구분for문while문
초기값 for문 내부에 포함됩니다 반복문 밖에서 작성됩니다
조건 검사 반복 시작 전 검사됩니다 반복 시작 전 검사됩니다
증감식 for문 내부에 포함됩니다 실행문 내부에서 직접 작성합니다
구조적 특징 반복 횟수가 명확할 때 사용됩니다 조건 중심 반복에 사용됩니다
