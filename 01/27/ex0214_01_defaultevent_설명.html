디폴트 이벤트(Default Event)

디폴트 이벤트란 이벤트가 발생했을 때 브라우저가 기본적으로 수행하는 동작을 말합니다.

  • <a> 태그 클릭 → 다른 페이지로 이동
  • <form> submit → 서버로 데이터 전송

이 기본 동작을 막고 싶을 때 디폴트 이벤트 제거를 사용합니다.

디폴트 이벤트 제거 방법

방법 1 : onsubmit에서 false 반환


document.getElementById("my-form").onsubmit = function(){
  return false; // submit 기본 동작 제거
};

방법 2 : 인라인 이벤트에서 return false


function whenSubmit(){
  return false;
}

<form onsubmit="return whenSubmit()">

방법 3 : event.preventDefault()


<a href="http://www.naver.com" 
   onclick="event.preventDefault()">
네이버 이동 막기
</a>

방법 4 : addEventListener + preventDefault


function whenSubmit(event){
  event.preventDefault(); // 기본 submit 동작 제거
}

document.querySelector("#login")
        .addEventListener("submit", whenSubmit);

정리

  • 디폴트 이벤트는 브라우저가 자동 수행하는 기본 동작입니다.
  • return false 또는 event.preventDefault() 로 제거합니다.
  • 폼 검증, AJAX 전송 등에 반드시 활용됩니다.
