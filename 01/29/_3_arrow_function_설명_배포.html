JavaScript 애로우 함수 (Arrow Function) 교안

애로우 함수(Arrow Function)는 ES6에서 도입된 함수 표현식 문법으로, 짧고 간결한 형태로 함수를 작성할 수 있도록 제공됩니다.

1. 애로우 함수 기본 문법

(매개변수 리스트) => { 실행할 문장들 }

함수 선언문 형태로는 사용할 수 없으며, 함수 표현식으로만 정의됩니다.

// 일반 함수 표현식
const sayHi = function(user) {
  console.log("Hi, " + user + "!");
};

// 애로우 함수 표현식
const sayHi2 = (user) => {
  console.log(`Hi, ${user}!`);
};

sayHi("arrow1 function");
sayHi2("arrow2 function");

2. 매개변수 규칙

① 매개변수가 한 개인 경우 소괄호는 생략 가능합니다.

user => { console.log(user); }

② 매개변수가 없는 경우 소괄호는 반드시 작성해야 합니다.

() => { console.log("Hello"); }

③ 매개변수가 여러 개인 경우 소괄호를 사용해야 합니다.

(a, b) => { return a + b; }

3. 실행문이 한 줄일 때

실행문이 한 줄이면 중괄호와 return을 생략할 수 있습니다.

// 기본 형태
const add1 = (a, b) => {
  return a + b;
};

// 생략 형태
const add2 = (a, b) => a + b;

console.log(add2(5, 3)); // 8

4. 애로우 함수와 this 차이

일반 함수는 호출한 객체에 따라 this가 결정됩니다. 하지만 애로우 함수는 자신만의 this를 가지지 않고, 상위 스코프의 this를 그대로 사용합니다. 

const person = {
  name: "홍길동",
  sayHi: function() {
    setTimeout(function() {
      console.log(window.name ===  this.name); //true
    }, 500);
  }
};

const person2 = {
  name: "이순신",
  sayHi: function() {
    setTimeout(() => {
      console.log(window.name ===  this.name); //false
      console.log(this.name); // "이순신"
    }, 500);
  }
};

person.sayHi();
person2.sayHi();;
콜백 함수 안에서 this를 유지하려면 애로우 함수가 유용하게 사용됩니다.

5. 최종 정리

• 애로우 함수는 간결한 함수 표현식 문법입니다.
• function 키워드 대신 => 를 사용합니다.
• 매개변수 규칙에 따라 괄호 생략이 가능합니다.
• 한 줄 실행문은 return 생략이 가능합니다.
• 애로우 함수는 자체 this를 가지지 않습니다.
